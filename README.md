# MTS-Teta-Project
Mobile app project for MTS Summer School 2021 (MTS.TETA)
- Clean Architecture
- MVVM
- Room, Retrofit
- Coroutines + LiveData
- Hilt
- Jetpack Navigation
- Transitions API, Motion Layout
